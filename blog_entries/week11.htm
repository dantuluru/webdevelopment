In my project, I will be using the following main Javascript libraries

  • jquery
  • jquery-ui: To create tabs
  • leaflet.js: To show map
  • jqPlot: To create plot

I will be using the following api's

  • fcc.gov api to get block, county, state information
  • census.gov api to get various metrics based on block, county, state information

When a user clicks on the map, we get the county and state information by using the latitude and longitude to query the fcc.gov api. The county and state information is then used to query the census.gov api to get the population for different races in the county. The data is then plotted as a bar chart.
